## Dia 1 (14/05/2026)

# Preparação do Ambiente e Instalações Iniciais

## 1. Objetivo do Dia

O primeiro dia do projeto teve como foco a preparação do ambiente físico e lógico da infraestrutura que será utilizada durante o Projeto Integrador.

Foram separados os equipamentos, identificados os dispositivos, iniciadas instalações dos sistemas operacionais e realizadas as primeiras configurações da rede.


---

## 2. Equipamentos Utilizados

Durante esta etapa foram utilizados os seguintes equipamentos:

3 computadores servidores

2 notebooks

1 roteador D-Link Dual Band

6 cabos de rede

Monitores para os servidores

2 monitores auxiliares para os notebooks



---

## 3. Estrutura Inicial do Projeto

A infraestrutura do projeto foi planejada da seguinte forma:

|-------------|--------|
| Equipamento | Função |
|Servidor Debian 01|Firewall e recebimento da internet|
|Servidor Debian 02|Servidor Tomcat, MySQL e Java|
|Windows Server 2025| 	Serviços Windows Server|
|Roteador D-Link Dual Band|Access Point da rede|
|Notebook 01|	Windows 10|
|Notebook 02|	Windows 10|
|----------------------|

---

## 4. Explicação dos Serviços Utilizados

**Firewall**

O firewall será responsável por controlar o tráfego da rede, permitindo maior segurança entre os dispositivos e a internet.

**Tomcat**

O Apache Tomcat é um servidor utilizado para executar aplicações Java na web.

**MySQL**

O MySQL é um sistema de banco de dados utilizado para armazenar informações das aplicações.

**Java**

O Java será utilizado para executar aplicações compatíveis com o servidor Tomcat.

**Access Point**

O roteador configurado como Access Point será responsável por distribuir a conexão de rede para os dispositivos do ambiente.

---

## 5. Atividades Realizadas

Durante o primeiro dia foram realizadas as seguintes atividades:

- Separação dos equipamentos necessários para o projeto;
- Organização física da bancada;
- Identificação dos hardwares utilizando etiquetas;
- Identificação dos cabos de rede;
- Início da instalação do Windows 10 nos notebooks;
- Configuração inicial do roteador D-Link;
- Início da instalação do Debian Server;
- Instalação inicial do Tomcat, MySQL e Java;
- Atualização do Debian finalizada;
- Início da instalação do Windows Server 2025;
- Finalização da instalação do Windows Server 2025.

## 7. Observações Técnicas

Durante a montagem inicial do ambiente, os equipamentos e cabos foram identificados com etiquetas para facilitar:

- organização da infraestrutura;
- identificação rápida dos dispositivos;
- manutenção futura;
- redução de erros de conexão.

Essa prática é muito utilizada em ambientes corporativos e datacenters.
